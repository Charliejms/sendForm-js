


# Instruciones:

1. **Paso 1** Clonar el repo e isntalar los modulos necesarios:
  ```
  npm install
  ```
  o
   ```
    npm i
   ```


2. **Paso 2** Ejecutar el Backend:
  ```
  npm run backend
  ```

3. **Paso 3** Ejecutar Gulp
  ```
  gulp
  ```
